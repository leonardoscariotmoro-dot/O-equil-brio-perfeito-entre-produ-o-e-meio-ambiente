# Agro Forte, Futuro Sustentável o equilíbrio perfeito entre produção e meio ambiente

Objetivo do Projeto

Demonstrar como as praticas de rotatividades de cultural de um sistema agroecologico, é de extremamente eficaz para efetivar o aumento da produtividade e preservação ambiental.

Recursos Técnicos
-HUB DE ACESSIBILIDADE:
O projeto conta com um painel de controle exclusivo que permite:
-controle de fonte: Ajuste dinâmico do tmanho do texto;
-Dark Mode Nativo: Troca de variáveis de cores CSS;

TECNOLOGIAS UTILIZADA

-Estrutura:HTML;
-Estilização:Css;
-Interatividade:JavaScript;
-Imagens: Internet;
